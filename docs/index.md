## 安装依赖

```bash

$ npm install --save antd mobx mobx-react-lite moment

```

$ npm install --save @alifd/next mobx mobx-react-lite moment

```bash
$ npm install --save @formily/antd @formily/react

```

or

```bash
$ npm install --save @formily/next @formily/react

```
